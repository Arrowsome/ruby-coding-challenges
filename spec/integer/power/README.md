# Power

## Instructions

Given positive integer `base` and positive integer `exponent` implement a function which calculates power of base of
exponent.

## Limitations

Do not use Ruby's ** operator.

[Challenge](challenge_spec.rb) | [Solution](solution.rb)

## Examples

```ruby
power(2, 1) # 1

power(2, 2) # 4

power(3, 3) # 27
```
