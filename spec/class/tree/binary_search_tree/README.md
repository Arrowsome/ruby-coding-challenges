# Binary search tree

## Instructions

Implement [binary search tree (BST)](https://en.wikipedia.org/wiki/Binary_search_tree).

[Challenge](challenge_spec.rb) | [Solution](solution.rb)

## Examples

```ruby
#----------Tree------------
#
#           6
#         /   \
#        3     8
#       /
#      1   
#
#--------------------------

bst = BinarySearchTree.new
bst.add(6)
bst.add(3)
bst.add(1)
bst.add(8)
```
