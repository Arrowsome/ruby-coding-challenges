# Product

## Instructions

Given an array of integers implement a function which returns the product of all numbers in the array.

[Challenge](challenge_spec.rb) | [Solution](solution.rb)

## Examples

```ruby
product([1, 2]) # 2

product([1, 2, 4]) # 8

product([2, 4, 10]) # 80
```
